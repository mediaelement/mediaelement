Stretching: 'auto'

Since the width="100%" height="100%" attributes is not set in the first video player tag, nothing is going to happen (stretching is 'none')...


... But in this one those attributes are set, so a responsive behavior will be set.

Stretching: 'responsive'

Doesn't matter if width="100%" height="100%" attributes are set, player will be responsive

Stretching: 'fill'

Special size: mostly used for background videos; it all depends of the dimensions of the parent container
